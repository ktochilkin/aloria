## Песочница на движке (главный козырь — этого нет у конкурентов)

- ⭐🔧 **eat-the-book** — слайдер объёма рыночной заявки, уровни стакана гаснут сверху вниз, средняя цена и слиппедж считаются вживую. *(предложили 5 линз — сильнейший сигнал)* · reading-market/ликвидность · wow 5 / effort 2.
- ⭐🔧 **limit-or-wait / fill-or-no-fill** — двигаешь цену лимитки по стакану; выше лучшего аска → мгновенно исполнилась, ниже → встала «Активна» и ждёт. *(5 линз)* · first-trade/order_basics · wow 5 / effort 3.
- ⭐🔧 **market-maker-reveal** — тумблеры «показать маркетмейкера / заявки учеников», строки стакана раскрашиваются по источнику. Видно, кто держит спред. · reading-market · wow 5 / effort 4.
- ⭐🔧 **macro-shock-candles** — карточки событий (ставка↑, отчёт, санкции): сначала предсказываешь стрелкой, потом тапаешь — свечи достраиваются, стакан перекашивается. · stocks/reading-market · wow 5 / effort 4.
- ⭐🔧 **live-instrument-peek / action-leaves-trace** — дип-линк в живой стакан реального тикера прямо из урока; направляемый сценарий заявка→сделка→лента→позиция. · reading-market · wow 5 / effort 4.
- 🔧 **matching-engine-mini / marketplace-match** — два слайдера (цена покупателя и продавца), сводишь навстречу → искра «Сделка», принт в ленту. · first-trade/exchange · wow 4 / effort 2.
- 🔧 **two-students-one-book** — сплит-экран: ты и другой ученик в одном стакане, заявки встречаются. Рынок Aloria реально общий. · reading-market · wow 4.
- 🔧 **order-status-journey** — выбираешь сценарий (есть встречная / мало ПС / отменил), чип статуса идёт working→filled/canceled/rejected с человеческим объяснением. · first-trade · wow 4 / effort 2.
- 🔧 **buying-power-meter** — слайдер объёма, шкала покупательной способности тает, кнопка «купить» гаснет при превышении («не хватает ПС»). · first-trade · wow 3 / effort 2.
- 🔧 **order-queue-priority** — твоя лимитка в очереди по цене/времени; встречные сделки съедают объём перед ней. Почему лимитка по лучшей цене может ждать. · reading-market · wow 5 / effort 4.
- 🔧 **predict-the-print** — тапаешь, по какому уровню пройдёт твоя рыночная покупка; движок проигрывает мэтчинг («ждал last, платишь по ask»). · reading-market · wow 4.
- 🔧 **fill-or-partial** — объём заявки больше, чем в стакане → «исполнена N из M», остаток повис. Понятие частичного fill. · first-trade · wow 3.
- 🔧 **spread-cost-roundtrip** — «купить по рынку» и сразу «продать по рынку», счётчик «потеряно на спреде»; сравнение ликвид/неликвид. · reading-market/spread · wow 4 / effort 2.
- 🔧 **depth-imbalance-meter** — соотношение объёмов bid/ask как индикатор давления (честно: «сигнал, не предсказание»). · reading-market · wow 3.
- 🔧 **book-vs-tape-mirror** — стакан слева, лента справа: уровень убывает → зеркально появляется строка ленты. Намерение → факт. · reading-market/orderbook · wow 4.
- 🔧 **market-order-eats-book** — рыночная проедает стакан вглубь, подсветка съеденных уровней. · reading-market · wow 5 / effort 4.
